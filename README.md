# Password Generator (Python)

A simple command-line password generator built using Python. This tool allows users to generate secure and customizable passwords based on length and character preferences.

---

## Features
- Generate random passwords of any length  
- Option to include:
  - Uppercase letters  
  - Lowercase letters  
  - Numbers  
  - Special characters  
- Simple and interactive CLI interface  


## Technologies Used
- Python  
- Built-in libraries: `random`, `string`  

---

## How to Run

```bash
python password_generator.py

=== Password Generator ===
